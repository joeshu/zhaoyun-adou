# 《赵云与阿斗》v2 工程设计（按全量资料落地）

375×667 竖屏 Canvas，零依赖。逻辑与渲染分层，core 部分 node 可无头跑（smoke.js）。

## 模块划分（script 顺序加载，共享全局）

| 文件 | 职责 |
|---|---|
| js/data.js | 布局常量、兵种/武将/羁绊/装备/道具/怪物/BOSS/彩蛋 全量数据表、通用工具 |
| js/save.js | 持久化：金币、材料、主线进度、武器仓、穿戴、道具拥有/携带、彩蛋、无尽 |
| js/merge.js | 合成算法（同型同级二合一、武将两字按序拼合）、unitStats（品级/等级/装备/羁绊）、回收返还 |
| js/battle.js | 战斗模拟：怪物寻路+攻击单位、单位攻击/克制、武将技能、专武特效、灵蛇、压力怪、阿斗 |
| js/waves.js | 波次生成：关卡缩放、特殊敌人解锁表、BOSS 排程（张梁/重甲骑兵/渠帅） |
| js/actions.js | 征兵抽卡入合成栏、拖拽落子（栏内合成/部署/撤回/铲子/回收）、主动道具、AI 镜像操作 |
| js/game.js | 对局状态机、经济 tick、结算奖励、彩蛋触发、无尽模式 |
| js/ui.js | 三段式渲染 + 输入 + 菜单/商店/锻造/装备 界面 |

## 布局（三段式）

- 顶栏 0–32：馒头 | 金币 | 关卡·波次 | 速度/暂停/菜单
- 敌方战区 32–296：敌阿斗(187,58)，敌棋盘 3×5，敌路径 4 横道蛇形
- 我方战区 300–532：我棋盘 3×5（行 y=352/408/464），路径 4 横道，阿斗(187,536)
- 底部 532–667：合成栏 2×5（10 格）+ 征兵按钮 + 主动道具×2 + 回收站

## 核心规则映射

- 征兵 → 随机产物进合成栏（兵种高权重 / 武将字低权重·招贤令加成 / 铲子）；费用 10+2×次数
- 合成栏：同型同级两两合升品（白绿蓝紫橙，战力 ×1/2/4/8/16）；拖 src 到 dst 且 `dst字+src字==武将名` 才成将（按正确顺序）
- 部署：栏 ↔ 棋盘 双向拖拽；铲子开荒地格；回收站返还馒头（垃圾桶道具 ×1.5）
- 单位有 HP，怪会停下围殴前排（盾嘲讽、甲减伤），防线破了才漏到阿斗；阿斗 3 血（大补丸+2，彩蛋配饰+1，续命丹免死一次）
- 武将 12 位（橙6紫6），击杀升级至 5 级，技能自动释放；羁绊：桃园/五虎/父子
- 装备：锻造（50金+2材料）随机出蓝/紫/橙；专武特效（龙胆飞枪、蛇矛灵蛇、落日弓距离增伤等）；穿戴按武器系匹配，橙武锁定专属武将
- 道具：6 携带位（主动≤2），主动战斗内按钮释放（陨石/神兵符/攻速符/毛笔/降妖符/淤泥），被动自动生效（农民/招贤令/洛阳铲/垃圾桶/大补丸/续命丹）
- 双人对抗：AI 镜像同刷同操作；击杀数每满 10 给对方生成压力怪；先死阿斗者输
- 关卡 1–30：波次无限刷至分胜负，强度随关卡+波次缩放；第6波张梁（摄魂全场晕）、10/20/30 关第8波重甲骑兵、周期渠帅；弩(≥10关)/斧(≥10关)/狂(≥13关)
- 彩蛋：15关点"旗"、28关触"藤"、30关结算停留10秒得阿斗配饰；全收集得限定皮肤
- 30 关通关解锁无尽模式（记录最高波次）
