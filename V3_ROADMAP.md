# V3 内容扩展实施清单

## 已确认范围
- 菜单信息架构收束与战斗 UI 安全区
- 主将战斗识别、返场反馈、将魂共鸣表现
- 局内军令、军师选择、Boss 预警/对策
- 主线地图机制、羁绊组合技
- 五虎试炼节点扩展、赤壁火攻互动、护送分路、残局每日化、讨伐阶段化
- 群英谱、英雄挑战、日常/周常与挑战码

## 分层落地原则
1. 先复用现有战斗内核，只添加统一配置和状态，避免复制战斗循环。
2. 所有新增奖励统一进入 `SAVE`，并保留旧存档迁移。
3. 每个系统必须增加无头测试，再接入 UI。

## 批次
- A：军令、军师、Boss 预警、主将状态、菜单收束。
- B：模式节点、地图机制、羁绊组合技。
- C：群英谱、挑战、日/周循环、挑战码。
